了解字符串
----------
.. code-block:: python

    'Hello!' * 5  # 结果是什么

    str(float()) * 2  # 结果是什么？

    a = '天高地厚'
    b = '天高' + '地厚'
    id(a) == id(b)  # 内存地址是否相同，相同就说明是同一个对象，否则就只是内容相同的两个对象

    c, d = '天高地厚', '天高地厚'
    id(c) == id(d)
    id(c) == id(d) == id(a)  # 结果如何？
    # 所以，同时赋值和单个独立赋值有什么优势？
