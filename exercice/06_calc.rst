x += y 和 x = x + y
-------------------
.. code-block:: python

    >>> x=1;x += 1;print x
    2
    >>> x=1;x = x+1;print x
    2
    >>> x=[1];x+=[2];print x
    [1, 2]
    >>> x=[1];x=x+[2];print x
    [1, 2]

    >>> x=[1]; print id(x);x=x+[2];print id(x)
    4357132800
    4357132728
    >>> x=[1]; print id(x);x+=[2];print id(x)
    4357132800
    4357132800

    # +与+=
    # +不能用来连接列表和元祖，而+=可以（通过iadd实现，内部实现方式为extends(),所以可以增加元组），+会创建新对象
    # 不可变对象没有__iadd__方法，所以直接使用的是__add__方法，因此元组可以使用 += 进行元祖之间的相加
