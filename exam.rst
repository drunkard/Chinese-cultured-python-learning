试题
====
拿试题练习有助于更好地理解 Python 编程。


True or False ?
---------------
每行一个题，判断真假。

.. code-block:: python

   1 == True
   2 == True
   0 == False

   1 is True
   0 is False

   True + True
   True + False
   True + True * False
   type(True)
   False * True

   -(-1) == 1
   len(bin(5))  # 值是多少？
   complex(1, 2) == complex('1+2j')  # 虚数
   int(-22.9)  # 值是多少？
   str(float()) * 2  # 结果是什么？

   [x for x in range(10) if x % 2]  # 要干什么？
   [x for x in range(10) if not x % 2]  # 要干什么？

数字操作技巧
------------
数字比较
````````
.. code-block:: python

   a = 5
   2 < a < 8
   1 == a < 3
   a >= 5


二进制
``````
.. code-block:: python

   # Linux 下常见目录权限位
   0b111, 0b101, 0b101  # 755, 目录
   0b110, 0b100, 0b100  # 644, 普通文件

   0b111, 0b101, 0b000  # 750, /home 下的用户主目录
   0b100, 0b100, 0b100  # 444, 只读文件

   0b101, 0b101, 0b101  # 555, /bin 下的文件
   0b101, 0b111, 0b111, 0b111  # 5777, /tmp 的权限位


字典操作技巧
------------
.. code-block:: python

   # 两个字典

   {**dict1, **dict2}
